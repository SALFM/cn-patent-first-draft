# 《中华人民共和国专利法（2020年修正）》初稿生成相关规则摘要

来源文件：`references/中华人民共和国专利法_20201017.docx`

总结目标：提取《中华人民共和国专利法（2020年修正）》中与中国发明、实用新型专利初稿生成直接相关的上位规则。本文主要服务发明和实用新型初稿生成；外观设计仅作为类型边界提示。

## 一、发明创造类型

《专利法》第二条规定，发明创造包括三类：

1. 发明。
2. 实用新型。
3. 外观设计。

对本 skill 的影响：

- 发明和实用新型属于技术方案类专利，是初稿生成的主要服务对象。
- 外观设计保护产品整体或局部的形状、图案、色彩等形成的富有美感并适于工业应用的新设计；其文本重点是图片、照片和简要说明，不应混入发明/实用新型说明书和权利要求的生成逻辑。
- 当用户输入主要是外观、图案、色彩、视觉识别、平面标识设计时，应提示其更接近外观设计边界，而不是直接按发明或实用新型生成。

## 二、发明和实用新型定义

### 1. 发明

第二条规定，发明是指对产品、方法或者其改进所提出的新的技术方案。

初稿生成规则：

- 发明可以覆盖产品、方法及其改进。
- 方法流程、制造工艺、控制方法、处理步骤、系统运行方法、材料制备方法等，更通常应按发明方向判断。
- 初稿必须围绕“技术方案”展开，不能只写目标、商业模式、管理规则、抽象想法或效果愿望。

### 2. 实用新型

第二条规定，实用新型是指对产品的形状、构造或者其结合所提出的适于实用的新的技术方案。

初稿生成规则：

- 实用新型必须落在“产品”的形状、构造或者二者结合上。
- 实用新型初稿应重点写清楚结构组成、连接关系、位置关系、形状配合、工作原理和可实现效果。
- 纯方法、流程、算法规则、材料本身、用途、商业逻辑等，不宜判断为实用新型。

## 三、不授予专利权的情形

### 1. 违反法律、社会公德或公共利益

第五条规定，对违反法律、社会公德或者妨害公共利益的发明创造，不授予专利权。

初稿生成规则：

- 若输入方案明显用于违法目的、危害公共安全、损害公共利益或违背社会公德，应停止按可授权专利正常生成，并输出风险提示。
- 若方案用途本身中性，但存在高风险使用场景，应尽量将初稿限定在合法、正当、技术性的应用场景。

### 2. 遗传资源违法获取或利用

第五条规定，对违反法律、行政法规获取或者利用遗传资源，并依赖该遗传资源完成的发明创造，不授予专利权。

第二十六条还要求，依赖遗传资源完成的发明创造，申请人应在申请文件中说明直接来源和原始来源；无法说明原始来源的，应陈述理由。

初稿生成规则：

- 生物、医药、农业、基因、菌株、动植物资源相关方案，应检查是否依赖遗传资源。
- 若依赖遗传资源，应提示补充遗传资源的直接来源、原始来源或无法说明原始来源的理由。

### 3. 第二十五条列举的不授权客体

第二十五条规定，下列各项不授予专利权：

- 科学发现。
- 智力活动的规则和方法。
- 疾病的诊断和治疗方法。
- 动物和植物品种。
- 原子核变换方法以及用原子核变换方法获得的物质。
- 对平面印刷品的图案、色彩或者二者的结合作出的主要起标识作用的设计。

同时，第二十五条规定，动物和植物品种的生产方法，可以依照专利法规定授予专利权。

初稿生成规则：

- 发现自然规律、自然物质或天然现象本身，不应作为发明/实用新型方案生成。
- 纯数学方法、游戏规则、商业规则、管理规则、教学规则、抽象算法或人为约定，属于高风险客体；只有当其落实为解决技术问题的具体技术方案时，才可继续按发明方向审视。
- 疾病诊断和治疗方法不授权；但医疗器械、检测设备、药物产品、制备方法等，应进一步判断是否属于产品或方法技术方案。
- 动物和植物品种本身不授权；其生产方法可继续按发明方向判断。
- 平面印刷品中主要起标识作用的图案、色彩或组合，不应按发明/实用新型生成。

## 四、授权条件

第二十二条规定，授予专利权的发明和实用新型应当具备新颖性、创造性和实用性。

### 1. 新颖性

新颖性是指：

- 该发明或者实用新型不属于现有技术。
- 没有任何单位或者个人就同样的发明或者实用新型在申请日以前向国务院专利行政部门提出过申请，并记载在申请日以后公布或公告的专利文件中。

现有技术是指申请日以前在国内外为公众所知的技术。

初稿生成规则：

- 初稿应尽量提炼区别于现有技术的技术特征。
- 背景技术不能把申请人的核心创新完整公开成现有技术。
- 权利要求应围绕区别技术特征组织，避免只写常规目标或已知结构。

### 2. 创造性

第二十二条对发明和实用新型设定了不同创造性表述：

- 发明：与现有技术相比，应具有突出的实质性特点和显著的进步。
- 实用新型：与现有技术相比，应具有实质性特点和进步。

初稿生成规则：

- 发明的创造性标准高于实用新型，初稿中应更充分写明技术问题、区别特征、技术效果以及非显而易见的改进逻辑。
- 实用新型也必须体现相对于现有结构的实质性特点和进步，不能只是简单替换名称、常规拼接或无技术效果的外形变化。
- 说明书的发明内容和有益效果应与权利要求的区别特征对应。

### 3. 实用性

实用性是指该发明或者实用新型能够制造或者使用，并且能够产生积极效果。

初稿生成规则：

- 初稿应写出方案如何制造、使用或运行。
- 实施例不能停留在愿景描述，应包含足以实现的结构、步骤、参数、连接关系或工作过程。
- 有益效果应是技术效果，且能由技术特征合理推出。

### 4. 不丧失新颖性的例外

第二十四条规定，申请日前六个月内，因特定公共利益首次公开、政府主办或承认的国际展览会首次展出、规定学术会议或技术会议首次发表、他人未经同意泄露等情形，不丧失新颖性。

初稿生成规则：

- 若用户提及已公开，应追问或提示公开时间、公开方式和是否属于法定例外。
- 不应默认任何公开都享有宽限期；宽限期只适用于法定情形。

## 五、申请文件组成和撰写要求

第二十六条规定，申请发明或者实用新型专利，应提交请求书、说明书及其摘要和权利要求书等文件。

### 1. 请求书

请求书应写明发明或者实用新型名称，发明人姓名，申请人姓名或者名称、地址，以及其他事项。

初稿生成规则：

- 生成初稿时至少应输出规范、清楚的专利名称。
- 发明人、申请人、地址等身份信息可作为待补充项，不宜虚构。

### 2. 说明书

说明书应对发明或者实用新型作出清楚、完整的说明，以所属技术领域的技术人员能够实现为准；必要时应有附图。摘要应简要说明技术要点。

初稿生成规则：

- 说明书应包括技术领域、背景技术、发明内容/实用新型内容、附图说明、具体实施方式等常见模块。
- 说明书必须清楚、完整，达到本领域技术人员能够实现的程度。
- 对实用新型，附图通常对结构公开非常重要，应提示用户补充结构图、连接关系图、剖视图或爆炸图等。
- 摘要只概括技术要点，不应扩展出说明书和权利要求没有支持的新内容。

### 3. 权利要求书

权利要求书应以说明书为依据，清楚、简要地限定要求专利保护的范围。

第六十四条还规定，发明或者实用新型专利权的保护范围以权利要求的内容为准，说明书及附图可以用于解释权利要求。

初稿生成规则：

- 权利要求是保护范围核心，应优先保证清楚、简要、有说明书支持。
- 独立权利要求应包含解决技术问题所必需的技术特征。
- 从属权利要求应围绕进一步限定、优选结构、优选步骤、参数范围、连接关系或应用场景展开。
- 说明书中的实施方式和附图标记应支撑权利要求，不应出现权利要求中没有公开依据的泛化特征。

## 六、其他影响初稿的规则

### 1. 优先权

第二十九条规定，发明或者实用新型自外国第一次提出专利申请之日起十二个月内，又在中国就相同主题提出申请的，可以依法享有优先权；自中国第一次提出专利申请之日起十二个月内，又向国务院专利行政部门就相同主题提出申请的，也可以享有优先权。

第三十条规定，要求发明、实用新型优先权的，应在申请时提出书面声明，并在第一次提出申请之日起十六个月内提交第一次申请文件副本；未声明或逾期未提交的，视为未要求优先权。

初稿生成规则：

- 若用户已有在先申请，应提示核对首次申请日、国家/地区、申请号、相同主题和十二个月期限。
- 初稿中不要擅自编造优先权信息；应列为待确认事项。

### 2. 单一性

第三十一条规定，一件发明或者实用新型专利申请应限于一项发明或者实用新型；属于一个总的发明构思的两项以上发明或者实用新型，可以作为一件申请提出。

初稿生成规则：

- 多个方案并列输入时，应判断是否属于一个总的发明构思。
- 若多个方案缺少共同或对应的特定技术特征，应提示可能需要分案或分别申请。

### 3. 修改不得超范围

第三十三条规定，申请人可以修改专利申请文件，但发明和实用新型申请文件的修改不得超出原说明书和权利要求书记载的范围。

初稿生成规则：

- 初稿阶段应尽量把关键替代方案、优选方案、参数范围、结构变形、等同连接方式和可选实施例写入说明书，为后续修改留出原始记载基础。
- 后续改写不能新增原始说明书和权利要求未记载的技术特征或技术方案。

### 4. 同样发明创造只能授予一项专利权

第九条规定，同样的发明创造只能授予一项专利权。

同时，同一申请人同日对同样的发明创造既申请实用新型又申请发明，先获得的实用新型专利权尚未终止，且申请人声明放弃该实用新型专利权的，可以授予发明专利权。

两个以上申请人分别就同样发明创造申请专利的，专利权授予最先申请的人。

初稿生成规则：

- 对同一技术方案，不应建议重复获得两项有效专利权。
- 对结构型产品方案，可提示“同日申请发明和实用新型”是可能路径，但最终同样发明创造只能保留一项授权。
- 应提示尽早申请，避免同样发明创造被他人先申请。

### 5. 在中国完成的发明或实用新型向外国申请前的保密审查

第十九条规定，任何单位或者个人将在中国完成的发明或者实用新型向外国申请专利，应事先报经国务院专利行政部门进行保密审查；违反该规定向外国申请的，在中国申请专利不授予专利权。

初稿生成规则：

- 若用户提到中国完成、计划先向外国申请或同时进行国际申请，应提示保密审查要求。
- 该规则适用于发明和实用新型，不适用于外观设计。

### 6. 涉密发明创造

第四条规定，申请专利的发明创造涉及国家安全或者重大利益需要保密的，按国家有关规定办理。

初稿生成规则：

- 涉及国防、国家安全、重大公共利益、敏感基础设施等内容时，应提示保密申请或保密审查风险。

### 7. 诚实信用原则

第二十条规定，申请专利和行使专利权应遵循诚实信用原则，不得滥用专利权损害公共利益或者他人合法权益。

初稿生成规则：

- 不应生成明显虚假的技术效果、实验数据、实施例或优先权信息。
- 不应为明显滥用专利制度的目的构造不真实方案。

## 七、应写入 SKILL.md 的规则

- 本 skill 主要生成中国发明和实用新型专利初稿；外观设计仅作边界识别和提示。
- 生成前先判断专利类型：发明、实用新型、外观设计边界或不适合授予专利权的客体。
- 发明应是针对产品、方法或者其改进提出的新的技术方案。
- 实用新型应是针对产品形状、构造或者其结合提出的适于实用的新的技术方案；纯方法、流程、算法、材料本身、用途或商业规则不得按实用新型生成。
- 对违反法律、社会公德、妨害公共利益、违法获取或利用遗传资源的方案，应输出风险提示，不按普通可授权方案处理。
- 对科学发现、智力活动规则和方法、疾病诊断和治疗方法、动物和植物品种、原子核变换方法及相关物质、平面印刷品标识性设计，应进行不授权客体检查。
- 发明和实用新型初稿必须围绕新颖性、创造性、实用性组织内容，写清楚区别技术特征、解决的技术问题和产生的技术效果。
- 发明创造性要求“突出的实质性特点和显著的进步”；实用新型创造性要求“实质性特点和进步”。发明初稿应更充分展开技术贡献链条。
- 申请文件应至少覆盖请求书信息占位、说明书、摘要、权利要求书等组成；不得虚构申请人、发明人、优先权等身份或程序信息。
- 说明书应清楚、完整，使所属技术领域技术人员能够实现；必要时提示补充附图。
- 权利要求书应以说明书为依据，清楚、简要限定保护范围；保护范围以权利要求为准，说明书和附图用于解释权利要求。
- 多个技术方案应检查单一性；不属于一个总的发明构思时提示拆分申请。
- 初稿应尽量纳入关键替代方案、优选方案和实施例，避免后续修改因缺少原始记载而超范围。
- 对同一发明创造，应提示只能授予一项专利权；同一申请人同日申请发明和实用新型时，应提示后续放弃在先实用新型后才可能授予发明。
- 中国完成的发明或实用新型拟向外国申请前，应提示保密审查要求。

| 第三十三条 | 发明/实用新型修改不得超出原说明书和权利要求书记载范围 | 初稿阶段保留充分原始记载，后续修改防止超范围 |

## 九、应写入 self_checklist.md 的检查项

- 是否已判断主题属于发明、实用新型、外观设计边界或不授权客体。
- 若为发明，是否属于产品、方法或者其改进的技术方案。
- 若为实用新型，是否限定在产品的形状、构造或者其结合，且适于实用。
- 是否排查违反法律、社会公德、妨害公共利益的风险。
- 是否排查违法获取或利用遗传资源的风险；如涉及遗传资源，是否提示直接来源和原始来源。
- 是否排查科学发现、智力活动规则和方法、疾病诊断和治疗方法、动物和植物品种、原子核变换方法及相关物质、平面印刷品标识性设计。
- 是否写明最接近现有技术或背景问题，并避免把核心创新写成现有技术。
- 是否提炼区别技术特征。
- 是否说明该区别技术特征解决的技术问题。
- 是否说明由区别技术特征带来的技术效果。
- 是否分别满足新颖性、创造性、实用性的表达要求。
- 对发明，是否体现“突出的实质性特点和显著的进步”。
- 对实用新型，是否体现“实质性特点和进步”。
- 是否包含足以实施的结构、步骤、参数、连接关系、材料、工作过程或实施例。
- 说明书是否清楚、完整，达到本领域技术人员能够实现的程度。
- 权利要求是否以说明书为依据。
- 权利要求是否清楚、简要。
- 独立权利要求是否包含解决技术问题的必要技术特征。
- 从属权利要求是否提供有层次的优选限定。
- 摘要是否只概括技术要点，未加入无支持的新内容。
- 是否检查多个方案之间是否属于一个总的发明构思。
- 是否提示优先权信息：首次申请日、国家/地区、申请号、相同主题、十二个月期限和声明/副本要求。
- 是否提示同样发明创造只能授予一项专利权。
- 若同日申请发明和实用新型，是否提示后续可能需声明放弃已授权实用新型。
- 是否提示中国完成的发明或实用新型向外国申请前需保密审查。
- 后续修改或补充是否避免超出原说明书和权利要求书记载范围。
- 是否避免虚构发明人、申请人、实验数据、技术效果、优先权或实施例。

## 十、专利类型判断规则

1. 先判断是否属于不授权客体或禁止性情形。
   - 若方案违法、违背社会公德、妨害公共利益，或依赖违法获取/利用的遗传资源，输出不授权风险提示。
   - 若主题是科学发现、智力活动规则和方法、疾病诊断和治疗方法、动物和植物品种、原子核变换方法及相关物质、平面印刷品标识性设计，原则上不按发明/实用新型初稿生成；动物和植物品种的生产方法可继续按发明判断。

2. 再判断是否为外观设计边界。
   - 若创新点主要是产品整体或局部的形状、图案、色彩及其组合，且目的主要是美感或视觉识别，判断为外观设计边界。
   - 外观设计不使用发明/实用新型的权利要求和说明书生成逻辑。

3. 判断是否适合实用新型。
   - 若创新点是有形产品的形状、构造或者二者结合，并能通过结构特征表达，优先考虑实用新型。
   - 实用新型文本应围绕部件、连接关系、空间位置、形状配合、结构改进、工作原理和结构带来的技术效果。
   - 若核心创新无法落到产品结构，例如方法步骤、控制流程、算法规则、材料配方、制造工艺或用途，不判断为实用新型。

4. 判断是否适合发明。
   - 若创新点是产品、方法或者其改进形成的新的技术方案，可判断为发明。
   - 方法、流程、工艺、控制逻辑、系统运行过程、材料制备、检测方法、设备与方法结合方案，通常按发明处理。
   - 对计算机、软件、算法相关输入，只有在其体现为解决技术问题、采用技术手段并获得技术效果的具体方案时，才按发明方向继续组织初稿；纯规则或抽象算法应提示不授权风险。

5. 发明和实用新型都可能时的处理。
   - 若一个结构型产品既可作为发明也可作为实用新型，应根据用户目标、创造性高度、审查周期、保护策略判断。
   - 可提示同一申请人同日就同样发明创造申请发明和实用新型的路径，但必须说明同样发明创造最终只能授予一项专利权；若发明授权，需要在法定条件下放弃尚未终止的实用新型专利权。
